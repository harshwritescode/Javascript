  📌 JavaScript and Classes

🔹 OOP (Object-Oriented Programming)
JavaScript supports OOP concepts like objects, classes (syntactic sugar), inheritance, etc.

🔹 Object
An object is a collection of properties (data) and methods (functions)

Example:
{
  name: "Harsh",        // property
  age: 22,              // property
  greet: function() {   // method
    console.log("Hello");
  }
}

🔹 toLowerCase()
A built-in string method in JavaScript
It converts all characters of a string into lowercase

Example:
let str = "HELLO";
console.log(str.toLowerCase()); // "hello"
